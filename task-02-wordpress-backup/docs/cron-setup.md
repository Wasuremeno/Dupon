# Настройка автоматического запуска через cron

## Что такое cron?

Cron — это стандартный планировщик задач в Linux, который позволяет запускать программы и скрипты по расписанию.

## Базовая структура crontab
/путь/к/команде
│ │ │ │ │
│ │ │ │ └── День недели (0-7, где 0 и 7 = воскресенье)
│ │ │ └──── Месяц (1-12)
│ │ └────── День месяца (1-31)
│ └──────── Час (0-23)
└────────── Минута (0-59)


## Настройка бэкапа WordPress
