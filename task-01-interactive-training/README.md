# Задание 1: Интерактивный обучающий материал

## Задача
Преобразовать текстовую инструкцию по работе с CRM в интерактивный обучающий материал с визуальным сопровождением и проверкой знаний.


### Компоненты решения

| Компонент | Инструмент | Почему выбран |
|-----------|------------|---------------|
| База знаний | Notion | Удобная структура, встраиваемое видео, доступ по ссылке |
| Запись видео | Screen Studio | Автоматический zoom к курсору, проф. качество без монтажа |
| Тестирование | Google Forms | Бесплатно, просто встраивается, вебхуки |
| Автоматизация | n8n | Self-hosted, гибкая логика, визуальный интерфейс |
| Уведомления | Telegram Bot API | Мгновенно, удобно, можно делать интерактив |

### Детали реализации

#### 1. Структура страницы в Notion

[Пример структуры](./examples/notion-page-structure.md)

#### 2. Workflow в n8n

Готовый workflow для импорта: [crm-test-workflow.json](./n8n-workflows/crm-test-workflow.json)

**Логика:**
1. Получение данных из Google Forms (вебхук)
2. Проверка ответов
3. Если всё верно → отправка поздравления и ссылки на следующий модуль
4. Если ошибки → отправка списка тем для повторения со ссылками на видео

#### 3. Telegram-бот для вопросов (опционально)

Простой бот на Python, который отвечает на вопросы по документации:
- [bot.py](./telegram-bot/bot.py)
- Использует поиск по Notion через API

### Инструкция по развертыванию

1. **Notion:**
   - Создать страницу по шаблону
   - Записать видео через Screen Studio
   - Встроить видео в страницу

2. **Google Forms:**
   - Создать форму с вопросами
   - Настроить вебхук в n8n

3. **n8n:**
   - Импортировать workflow
   - Настроить токены Telegram и Google
   - Активировать

### Преимущества подхода

- **Масштабируемость:** легко добавить новые курсы
- **Self-hosted:** данные не уходят в облака (кроме Notion, но можно заменить на Confluence)
- **Гибкость:** n8n позволяет менять логику без программирования
- **Быстрота создания:** видео записываются за минуты

### Альтернативы

| Компонент | Альтернатива | Когда использовать |
|-----------|--------------|-------------------|
| База знаний | Confluence | Если уже есть экосистема Atlassian |
| Запись видео | Loom | Если нужна максимальная скорость |
| Автоматизация | Make.com | Если не хочется хостить самому |
| Тестирование | Typeform | Если нужен красивый дизайн |
